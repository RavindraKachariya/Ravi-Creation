# 🛍️ Ravi Creation - MERN E-Commerce Project

This is a **Full Stack MERN E-Commerce Application** built using **MongoDB, Express.js, React.js, and Node.js**.

The project includes:

* 🛒 **Customer Frontend Website**
* ⚙️ **Admin Dashboard**
* 🗄️ **Backend API Server**

Currently this repository is shared mainly for **Live Demo purposes**.
Full documentation will be added in future updates.

# 📁 Project Structure

The project is divided into **three main folders**:

```
project-root
│
├── Frontend   → Customer E-Commerce Website (React)
│
├── Admin      → Admin Dashboard (React)
│
└── Backend    → Node.js + Express API + MongoDB
```
